# Usage

Vapor 라이브러리의 [개발 Quick start](/guides/usage#quick-start) 및 문서화 사이트 [Vapor Docs 사용법](/guides/usage#docs-usage)을 안내합니다.

## Quick Start

[Vapor Installation](/guides/installation)을 완료하면 Vapor 라이브러리를 사용할 수 있습니다.

```javascript
import { Button, Input } from '@goorm-dev/vapor-components';
import { useTrapFocus } from '@goorm-dev/vapor-hooks';
import { SeachIcon } from '@goorm-dev/vapor-icons';

function App() {
    const ref = useRef < HTMLDivElement > null;
    useTrapFocus({ trapContainer: ref, onEscape });

    return (
        <div ref={ref} role="dialog">
            <Button color="primary">
                Click
                <SeachIcon />
            </Button>

            <Input type="text" placeholder="Enter your name" />
        </div>
    );
}
```

### TypeScript Support

Vapor는 TypeScript를 완벽히 지원합니다. TypeScript 프로젝트에서 Vapor 컴포넌트를 활용해보세요.

```typescript
import { ButtonProps } from '@goorm-dev/vapor-components';

type MyButtonProps = ButtonProps & {
    // Add custom props
};
```

#### Vapor Sync with Figma

Vapor 라이브러리들은 Figma와 동기화되어 있습니다. 개발시에 Goorm Figma에 정의된 컴포넌트를 활용해 시안 개발을 할 수 있습니다.
